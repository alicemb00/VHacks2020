Error-Correct a Reed-Muller Code!

Reed-Muller codes are a class of linear codes denoted by RM(r, m), where 0 ≤ r ≤ m. They are defined recursively and are used for deep-space communication!

Each word is encrpyted using a generator matrix, then sent. Error patterns of weight ⌊2m-r / 2⌋ can be corrected. If r = 1 and m = 3, then if a sent codeword has up to 2 bits flipped, it can be corrected!

Be sure to enter a word of length 2m.
